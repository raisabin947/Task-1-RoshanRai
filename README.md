# My Portfolio Website 💻

Hey! This is my personal portfolio website where I showcase my work, skills, and background as a Frontend Developer and BSc CSIT student. I wanted a place to put everything together—from my education timeline to my favorite projects and quick ways to get in touch with me.

## 📌 Just a Simple Static Site
There is no complicated backend, database, or heavy framework setup here. It is just a straightforward **static webpage** built from scratch using:
- **HTML5** to handle the text, layout structure, and sections.
- **CSS3** to make it look good with the dark theme, grid layout, glowing blue accents, and to make sure it looks clean on mobile phones.

## 📂 What's in the Box?
- `index.html` — The main file that holds all the text content, structure, and links.
- `style.css` — The stylesheet that controls the design, colors, fonts, and responsiveness.
- `profile.jpg` — My profile picture.

## 🚀 How to Open and Look at It
1. Download all the files and make sure `index.html`, `style.css`, and your profile picture are sitting inside the exact same folder.
2. Simply double-click on `index.html`. It will instantly open up and run right inside your favorite web browser!

## ✏️ Want to Tweak It?
If you want to change the text, update a skill percentage, or swap out a project name, just open the `index.html` file using Notepad or any code editor (like VS Code) and edit the text directly. It's super easy to update!
